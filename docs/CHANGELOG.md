# Changelog

## [1.0.0] - 2026-03-28

### Added
- Three-layer spatial architecture: Window, Volume, Immersive Space
- Planet data model with 8 planets' real astronomical data
- ECS orbit/rotation systems with testable math
- EntityFactory for programmatic 3D entity creation
- Sun corona particle effect
- 2K planet textures (CC-BY 4.0) with async loading and fallback
- Gesture interactions: tap, drag-rotate, pinch-zoom
- @MainActor isolation for Swift 6 concurrency
- 10 unit tests (PlanetTests + OrbitSystemTests)
